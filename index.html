Yogicfuture

A powerful, easy-to-use business platform

designed specifically for yoga studios and schools

Class & Session Management

  • Manage multiple classes from one place
  • Admin can manage their free time and schedule one-to-one (private) sessions
  • Admin can manage their practice session
  • Admin can manage their program
  • Admin can manage their workshop
  • Admin can manage the number of clients who can subscribe to programs, workshops, and practice sessions
  • Admin can add membership class from Monday to Sunday
  • Admin can change class-related settings

Membership & Tier Management

  • Admin can manage membership tier
  • Admin can add a member to the membership tier
  • Admin can grant attendance for membership classes to the member
  • Admin can pause and resume the membership

Payment & Finance

  • RazorPay and Stripe both payment supported
  • Admin can create a discount coupon with a minimum order value and a maximum discount value (either flat or percentage)
  • Admin can give the early bird discount
  • Admin can also use dynamic pricing feature
  • Client can show their previous transactions
  • Admin can add balance to the client wallet

Subscription & Booking

  • Client can subscribe to private sessions, programs, workshops, and practice sessions
  • Client can subscribe to programs
  • Admin can subscribe to programs, workshops, practice sessions, and membership tiers on behalf of the client
  • Admin can add the client
  • Admin can add client to the waitlist
  • When a class is booked, both the client and admin receive an email notification

User Access & Role Management

  • Admin can manage their team with role-based access

Reports & Analytics

  • Admin can generate various types of dynamic report
  • Admin can give scores and attendance to the client

Notifications & Communication

  • Admin can change the default email template
  • Admin can send push notifications to the client

Onboarding & Personalization

  • Admin and client have a smooth onboarding experience
  • Client can update their personal information
  • Admin can change class-related settings and personal information

Space & Availability

  • Admin can share their available space with others

Data Privacy & Compliance

  • Client data will not be used externally and will be deleted if the admin leaves
